# 产品设计规范 v1.0

## 1. 概述

本文档描述了产品设计的基础规范，包括视觉设计、交互规范和组件使用准则。所有产品设计人员和开发人员都应当遵循本规范，以确保产品体验的一致性。

## 2. 视觉设计

### 2.1 色彩系统

主色调采用深蓝色 #1e3a5f，辅色使用琥珀色 #d97706 作为强调。中性色包含从白色到深灰色的九个色阶，用于背景、边框和文字。

### 2.2 字体规范

标题使用 Inter 字体，正文内容使用 Noto Serif SC 字体。代码块使用等宽字体。字号从 12px 到 40px 共八个级别，对应不同的文本层级。

## 3. 组件规范

### 3.1 按钮

按钮有三种尺寸：小、中、大。主按钮使用深蓝色背景配白色文字，次要按钮使用白色背景配蓝色边框和文字，危险按钮使用红色系。

### 3.2 表单

表单控件统一高度为 40px，圆角 8px，边框默认灰色，聚焦时显示蓝色边框和浅蓝阴影。标签在控件上方，间距 8px。

## 4. 交互规范

所有操作都应有即时反馈。按钮点击后会有视觉反馈（变色、阴影变化），耗时超过 300ms 的操作需显示加载状态。表单提交前必须进行前端校验，并在错误字段附近显示错误提示。
